Regression Analysis

For my final project in STAT 410 - Linear Regression, I completed a full linear regression analysis in an attempt to figure out which variables oddsmakers use to create betting odds for NFL games. The analysis is in the form of a writeup which uses various methods including AIC variable selection and prediction intervals. Analysis was done in R. The linked github repo has all of the R code and PDF.
